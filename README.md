# Smart Image Renamer

Инструмент для пакетного переименования фотографий. Разработан специально для фотографов и ретушеров, которым важна скорость, порядок и безопасность данных.

## 🚀 Основные возможности
- **Гибкая нумерация:** Поддержка различных шаблонов (IMG_0001, Name_0001 и др.).
- **Безопасность:** Режим создания резервных копий (Backup) перед внесением изменений.
- **Интеллектуальная сортировка:** Использование "натуральной" сортировки (файл 2 идет перед 10).
- **Умный интерфейс:** Современный дизайн, темная/светлая тема и история операций.
- **Предпросмотр:** Возможность увидеть результат до начала переименования.

## 🛠 Установка
1. Убедитесь, что у вас установлен [Python](https://www.python.org/).
2. Скачайте все файлы проекта в одну папку.
3. Запустите файл `install.bat`, чтобы автоматически установить нужные библиотеки.

## 💡 Как пользоваться
1. Запустите программу файлом `smart_renamer.pyw`.
2. Выберите папку с фотографиями нажатием кнопки **«Обзор»**.
3. Настройте шаблон именования и префикс (если нужно).
4. Нажмите **«Превью»**, чтобы убедиться в правильности имен.
5. Нажмите **«Переименовать!»**.

---
*Программа автоматически сохраняет ваши настройки и историю операций в папку с приложением.*
